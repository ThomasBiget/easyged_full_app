# User stories

## User

### V1
- en tant que user je veux pouvoir m'authentifier
- en tant que user je veux pouvoir uploader une image ou un pdf de mes factures
- en tant que user je veux pouvoir visualiser la facture uploader
- en tant que user je veux pouvoir extraire les données de ces factures
- en tant que user je veux pouvoir accéder à ces informations dans mon espace
- en tant que user je veux pouvoir voir la liste de mes factures
- en tant que user je veux pouvoir vérifier les informations extraitent de ces factures
- en tant que user je veux pouvoir modifier les informations extraitent de ces factures si elles sont erronées
- en tant que user je veux pouvoir supprimer une facture
- en tant que user je veux pouvoir supprimer une ligne de facture


### V2
- en tant que user je veux pouvoir rechercher une facture par mot clé dans une barre de recherche
